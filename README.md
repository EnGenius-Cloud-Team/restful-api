# EnGenius Cloud RESTful API
Documentation: https://liveapi-console-dev.s3-us-west-2.amazonaws.com/engenius_cloud/falcon.html

## Requirements

* Python 2.x/3.x (Recommend 2.7/3.8)

## Installation

```console
$ pip install requests
```
